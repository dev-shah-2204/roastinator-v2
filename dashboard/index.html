What Roastinator brings to your server
Fun

Roastinator has a lot of fun commands like roasting, meme, dadjoke, image manipulation and much more. There are always more commands coming so you never get bored!

Roastinator allows you to easily moderate your server with commands that allow you to kick, ban, mute and warn people. Moreover, it allows you to delete messages in bulk!
